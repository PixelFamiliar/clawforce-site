Scanning & Certification

Tier 1 (Scanned) results return in under 30 seconds. The free browser-based scan runs entirely client-side.

Tier 2 (Verified) includes intent analysis comparing your SKILL.md claims against actual code behavior. Results are typically delivered within 1-2 hours.

Tier 3 (Certified) includes Docker sandbox execution, behavioral monitoring, and automated behavioral analysis. Results are delivered within 24 hours.

ClawForce currently scans skills written in Python and JavaScript/TypeScript, which covers the vast majority of the OpenClaw skill ecosystem.

We analyze .py, .js, .ts, .jsx, .tsx, .mjs, .cjs, and .sh files automatically. JSON, YAML, and TOML configuration files are also included in structural analysis.

A ClawForce certificate is a signed JSON document containing: the skill name and version, certification tier, risk score, scan timestamp, findings summary, a SHA-256 hash of the scanned code, and a cryptographic signature.

You also receive an embeddable HTML/SVG badge for your README, and the skill is listed in the public ClawForce Registry.

You receive a detailed report explaining every finding, its severity, the file and line where it was detected, and remediation guidance. You can fix the issues and re-submit for free (Tier 1) or at a discounted rate (Tier 2/3).

A failed scan does not appear in the public registry unless you explicitly opt in.

Tier 3 certification is a fully automated pipeline. Your skill is executed inside an isolated Docker sandbox while ClawForce monitors system calls, network activity, file access patterns, and runtime behavior. The sandbox execution trace is then compared against your SKILL.md declarations to flag any undeclared capabilities.

The combined analysis (static scan + intent verification + sandbox trace) is used to produce the final risk score and signed certificate. The entire process is automated and typically completes within 24 hours.

ClawForce vs. Traditional Tools

Traditional SAST tools are built for general application code. They flag subprocess.Popen as a vulnerability because most web apps don't need it. But for an agent skill designed to run shell commands, it's the core feature.

ClawForce is purpose-built for AI agent skills. We analyze intent: we compare a skill's declared capabilities (in SKILL.md) against what the code actually does. This catches undeclared network access, credential harvesting, and supply-chain attacks that SAST tools would either miss or flag incorrectly.

We also provide certification, not just scanning. A ClawForce certificate is a signed attestation that a skill has passed specific security checks at a given point in time.

Pricing & Billing

We're actively developing subscription plans for teams and platforms that need to certify skills at scale. If you're scanning more than 10 skills per month, contact us to discuss custom pricing.

Tier 1 (Scanned) is always free, with no limits on the number of scans.

Payments are processed securely through Stripe. When you purchase a Tier 2 or Tier 3 certification, you provide a GitHub repository URL, then complete checkout via Stripe. Results are delivered to the email address associated with your payment.

We do not store any payment information. All billing is handled by Stripe.
