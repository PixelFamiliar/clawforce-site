Three Layers of Trust

Every skill goes through progressively deeper analysis. Higher tiers mean higher trust.

🟢
Scanned
Automated static analysis catches known dangerous patterns, secrets, and obfuscation.
  • 50+ vulnerability patterns
  • Secret & credential detection
  • Obfuscation flagging
  • Dependency audit
  • AST-level code analysis
🔵
Verified
Everything in Scanned, plus intent analysis verifying the skill does what it claims.
  • All Scanned checks
  • SKILL.md claim validation
  • Behavior vs declaration match
  • Undeclared capability detection
  • Permission scope audit
🟡
Certified
Full pipeline: scan, intent analysis, dependency audit, and AI-powered semantic code review.
  • All Verified checks
  • AI semantic code review
  • Supply chain analysis
  • Signed JSON certificate
  • Embeddable trust badge
