MCP-Specific Analysis

Standard security scanners treat MCP servers as regular code. They miss the protocol-specific risks:

Tool description injection: MCP tools declare their capabilities in natural language descriptions. A malicious tool can embed instructions that override the agent's system prompt. ClawForce analyzes tool descriptions for injection vectors that would poison the agent's context.

Resource exposure: MCP resources can expose filesystem paths, database contents, or API data to the agent. ClawForce maps resource declarations against actual data access to flag over-exposure.

Prompt flow analysis: MCP tools can return content that gets injected into the agent's prompt chain. ClawForce traces the data flow from tool response through the agent's reasoning to detect smuggled instructions.
