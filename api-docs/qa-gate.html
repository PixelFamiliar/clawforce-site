QA Gate API

Submit agent-generated content for deterministic compliance validation. Returns a verdict with detailed linting results in under 50ms.

POST/v1/qa-gate

Request Body

{
  "agentId": "string"           // Your agent identifier
  "proposedAction": "string"   // PUBLISH_CONTENT | SEND_EMAIL | API_CALL
  "content": "string"          // The agent-generated content to lint
  "requireHITL": boolean       // Force human-in-the-loop approval
  "ruleset": ["string"]       // Array of lint rule IDs to apply
}

Response

{
  "verdict": "APPROVED",
  "piiDetected": false,
  "citationsVerified": true,
  "hallucinationScore": 0.02,
  "auditId": "clf_log_1740189072",
  "settlementTimestamp": "2026-02-21T19:51:12.000Z",
  "appliedRules": ["strict-pii-block", "factual-citation-req"],
  "lintResults": [
    { "rule": "strict-pii-block", "pass": true, "details": "No PII detected" },
    { "rule": "factual-citation-req", "pass": true, "details": "2/2 claims cited" }
  ]
}

Available Rulesets

strict-pii-block

Block content containing PII (emails, phones, addresses)

factual-citation-req

Require source citations for factual claims

brand-voice-check

Validate tone against your brand guidelines

legal-disclaimer-req

Ensure required disclaimers are present

casl-compliance

CASL anti-spam compliance for outbound comms

hallucination-detect

Flag potential AI hallucinations via cross-referencing
