Permission Scopes

Every AI agent skill declares what it can access. ClawForce audits these declarations and flags mismatches between claimed and actual behavior.

Why scopes matter

A skill that claims read:workspace but makes network calls is a red flag. ClawForce detects scope violations during scanning and flags them in the registry.

Filesystem Scopes

Low read:workspace
Read files in the current workspace directory. Most common scope for code analysis tools.
Read .ts, .py, .json files
Med write:workspace
Create or modify files in the current workspace. Required for code generation and formatting tools.
Write to src/, create new files
High read:filesystem
Read files anywhere on the filesystem, not just the workspace. Can access config files, SSH keys, and credentials.
Read ~/.ssh, ~/.env, /etc/passwd
Crit write:filesystem
Write files anywhere on the filesystem. Extremely dangerous, can modify system files or install backdoors.
Write to /usr/bin, ~/.bashrc

Network Scopes

Low net:localhost
Connect to services on localhost only. Common for database tools and local dev servers.
localhost:3000, 127.0.0.1:5432
Med net:allowlist
Connect to a declared list of external hosts. The skill specifies which domains it contacts.
api.github.com, api.openai.com
High net:outbound
Make arbitrary outbound network requests. Can exfiltrate data to any destination.
Any HTTP/HTTPS, WebSocket, TCP

Execution Scopes

Med exec:pre-approved
Run a pre-defined set of commands. The skill declares exactly which commands it will execute.
npm test, eslint, prettier
High exec:shell
Execute arbitrary shell commands. Can install packages, modify system state, or run scripts.
bash -c "...", sh, cmd.exe
Crit exec:sudo
Execute commands with elevated privileges. No legitimate skill needs this. Immediate rejection.
sudo, doas, runas

Data Scopes

Low data:context
Access the current conversation context only. Standard for most assistant tools.
Chat history, current prompt
High data:env
Access environment variables. Can read API keys, database credentials, and secrets.
$OPENAI_API_KEY, $DATABASE_URL
Crit data:credentials
Access stored credentials, keychains, or secret managers. Maximum risk scope.
macOS Keychain, SSH agent, GPG

How ClawForce uses scopes

During scanning, we analyze skill code to detect actual behavior and compare it against declared scopes. Mismatches are flagged as findings. Skills that access data:env without declaring it get a higher risk score. Skills with exec:sudo are automatically rejected.
