For AI Developers

Your agents need web data. Instead of getting blocked, rate-limited, or scraped into trouble, ClawForce gives you authenticated, paid access to any merchant on the network.

How it works

STEP 1
Register your agent

Create a developer account and register each AI agent. You'll get a unique API token per agent.

STEP 2
Fund your wallet

Add credit to your developer wallet via Stripe. Minimum $5. No subscription, no monthly fees. Your balance is deducted per request based on each merchant's pricing.

STEP 3
Make authenticated requests

Send your agent token with every request. ClawForce verifies the token, checks your balance, charges the per-request fee, and proxies the request to the merchant's site.

# Example request with agent token
curl -H "X-Claw-Agent-Token: claw_7f3a..." \
     https://example.com/api/data

# Response (proxied through ClawForce edge)
HTTP/1.1 200 OK
X-Claw-Charged: $0.001
X-Claw-Balance: $49.999

What you get

Guaranteed access. No more 403s. If the merchant accepts paid agents, your request goes through.

Transparent pricing. You see the per-request cost before you send. No surprises.

Clean audit trail. Every request, charge, and balance change is logged in your developer dashboard.

No rate limits. Pay-as-you-go means no artificial throttling. Send as many requests as your balance covers.

Security Tooling

CLI SCANNER
Scan skills from your terminal

22 vulnerability patterns across 6 categories. Zero dependencies. Works on Node 18+.

# Install globally
npm install -g clawforce

# Scan a local skill directory
clawforce scan ./my-skill

# JSON output for CI pipelines
clawforce scan . --json --threshold 20

# Get badge embed URLs
clawforce badge my-skill

Source: github.com/PixelFamiliar/clawforce-cli

GITHUB ACTION
Scan on every push and PR

Drop-in GitHub Action with configurable threshold, structured outputs, and automatic report artifacts.

# .github/workflows/security.yml
name: Security Scan
on: [push, pull_request]

jobs:
  scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: PixelFamiliar/clawforce-scan-action@v1
        with:
          fail-threshold: '40'

Source: github.com/PixelFamiliar/clawforce-scan-action

SDKs
Programmatic API access from any language

Zero-dependency client libraries for Node.js and Python. TypeScript types included.

# Node.js
npm install @clawforce/sdk

# Python
pip install clawforce
// Node.js usage
const { ClawForce } = require('@clawforce/sdk');
const client = new ClawForce();
const result = await client.scan('user/my-skill');
console.log(result.risk_score);  // 12

# Python usage
from clawforce import ClawForce
client = ClawForce()
result = client.scan('user/my-skill')
print(result.risk_score)  # 12
