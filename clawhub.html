ClawHub Skill Ecosystem

ClawHub aggregates AI agent skills from multiple sources. We scan every skill, generate SBOMs, score risk, and publish the results to the public registry.

What Gets Scanned

Every skill in ClawHub goes through the same pipeline regardless of source:

Prompt injection detection — Direct injection, roleplay/pretend patterns, context clearing, token smuggling

Malware signatures — Data exfiltration, arbitrary code execution, shell injection, reverse shells, crypto wallet theft

Unsafe data handling — Hardcoded secrets/credentials, sensitive data logging, API token leaks, private key exposure

Dependency analysis — Full transitive dependency tree, known CVEs, abandoned packages

Permission scope audit — Declared vs. actual behavior comparison (see Scopes Reference)
