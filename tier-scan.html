What Gets Scanned

🔍
Prompt Injection
Detects direct injection attempts, roleplay/pretend patterns, context clearing, and token smuggling in skill definitions.
🦠
Malware Signatures
Scans for data exfiltration patterns, arbitrary code execution, shell injection, reverse shells, and crypto wallet theft.
🔑
Data Handling
Identifies hardcoded secrets, sensitive data logging, API token leaks, and private key exposure in the codebase.
📋
Permission Analysis
Maps requested capabilities (filesystem, network, execution) and flags overly broad permission scopes.

What You Get

  • Risk score from 0-100 based on static analysis
  • Registry listing in the ClawForce Verified Skills database
  • SVG trust badge you can embed in your README
  • CycloneDX SBOM (Software Bill of Materials) for the skill
  • Content hash for identity verification
  • Public scan results anyone can verify

Limitations

The Scanned tier is fully automated. It catches common attack patterns but does not include:

  • Manual code review by security engineers
  • Behavioral analysis or runtime testing
  • Supply chain dependency auditing
  • Enterprise compliance documentation

For deeper analysis, consider the Verified or Certified tiers.
