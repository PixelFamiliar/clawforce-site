Secure Your Skills in 5 Minutes

From zero to scanned. Pick your integration point and you're running.

1

Install the CLI 30 seconds

The ClawForce CLI runs locally, scans files for 22 vulnerability patterns, and requires zero configuration.

# Install globally via npm
npm install -g clawforce

# Or run without installing
npx clawforce scan .
2

Scan your first skill 10 seconds

Point the scanner at any directory containing Python, JavaScript, TypeScript, or shell scripts.

# Scan the current directory
clawforce scan .

✓ ClawForce Scan Complete
Skill:       my-skill
Files:       8 scanned
Risk Score:  9/100 — PASS
Findings:    1 (medium)

# JSON output for programmatic use
clawforce scan . --json

# Set a threshold — exit code 1 if exceeded
clawforce scan . --threshold 20
3

Add to CI/CD 2 minutes

Add the GitHub Action to scan on every push and PR. Fails the build if risk exceeds your threshold.

# .github/workflows/security.yml
name: Security Scan
on: [push, pull_request]

jobs:
  scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: PixelFamiliar/clawforce-scan-action@v1
        with:
          fail-threshold: '40'
4

Add editor scanning 1 minute

Install the VSCode/Cursor extension for real-time scanning. Findings appear inline as you type.

# Clone and install
git clone https://github.com/PixelFamiliar/clawforce-vscode
# Open in VSCode → Run Extension (F5)

# Features:
# - Scan on save and open
# - Findings in Problems panel
# - Risk score in status bar
# - Configurable severity threshold
5

Get certified optional

Once your skill passes the automated scan, upgrade to Verified ($9) or Certified ($29) for manual review, a trust badge, and registry listing.

# Generate a badge URL
clawforce badge my-skill

Use the SDK

For programmatic access, use the Node.js or Python SDK:

// Node.js
const { ClawForce } = require('@clawforce/sdk');
const client = new ClawForce();
const result = await client.scan('user/my-skill');

if (result.risk_score > 40) {
  console.warn(`High risk: ${result.risk_score}/100`);
}

# Python
from clawforce import ClawForce
client = ClawForce()
result = client.scan('user/my-skill')

if not client.verify('my-skill').trusted:
    raise Exception('Untrusted skill')
